# 项目简介

**权限管理系统** 是一个基于 ASP.NET Core 10 + Vue 3 的全栈企业级管理平台，提供完整的用户认证、RBAC 角色权限、动态菜单、流程审批、入库单管理、低代码集成平台、OAuth 2.0/OIDC 单点登录、计划任务调度、外部数据库管理等功能。

## 核心特性

- **用户认证**：JWT + BCrypt 登录/注册，Redis Token 缓存与吊销，Cookie IdP 会话，在线用户追踪
- **RBAC 权限管理**：用户 → 角色 → 权限三层模型，22 项细粒度权限码，树形权限分配
- **动态菜单**：数据库驱动菜单树，按权限自动过滤，顶栏搜索（键盘导航），外部链接支持
- **密码重置**：邮箱验证码（SMTP/MailKit），6位数字码，5分钟有效期
- **用户管理**：分页/搜索/排序，CRUD，冻结/解冻，批量操作，在线状态，角色分配
- **流程审批**：LogicFlow 可视化拖拽建模，审批/抄送/条件分支，任签/会签策略，版本管理，超时自动处理
- **入库单管理**：完整采购入库流程，材料明细管理，附件上传（MinIO），CSV 导出，含税金额自动计算
- **材料字典**：500 条种子材料，10 大分类，关键字搜索，批量删除
- **统一附件**：MinIO 对象存储，按模块关联，上传/下载/删除
- **集成平台**：低代码数据同步，HTTP 连接管理（5 种认证方式），Pull→Transform→Push 流水线，字段映射，C# 脚本
- **计划任务**：Cron 定时调度，一次性任务，内置处理器（创建用户/清理日志/数据导出/统计），C# 脚本/反射处理器
- **外部数据库管理**：MySQL/SQL Server/SQLite/Oracle/PostgreSQL 五种数据库，表结构浏览，只读 SQL 执行
- **SSO 单点登录**：魔法链接（64位令牌），8位授权码，Redis 缓存验证
- **OAuth 2.0 / OIDC**：授权码流程 + PKCE S256，RS256 id_token，Refresh Token 轮转，OIDC Discovery，JWKS
- **主页仪表盘**：可配置卡片网格，天气/图表/统计/时钟/待办/已办 6 种卡片，用户级持久化
- **天气组件**：IP 自动定位（3 服务回退），wttr.in 数据，温湿度/风速/体感

## 内置账号

| 角色 | 用户名 | 密码 | 权限 |
|------|--------|------|------|
| 超级管理员 | `admin` | `password` | 全部 22 项 |

内置三种角色：超级管理员（全部权限）、普通用户（基础查看）、只读用户（仅查看）。
