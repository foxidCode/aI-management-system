# 项目简介

**权限管理系统** 是一个基于 ASP.NET Core 10 + Vue 3 的全栈企业级管理平台，提供完整的用户认证、角色权限管理、动态菜单、流程审批、入库单管理等功能。

## 核心特性

- **用户认证**：JWT 登录/注册，BCrypt 密码哈希，Redis Token 缓存，在线用户追踪
- **权限管理**：基于 RBAC 模型，用户 → 角色 → 权限三层控制，支持树形权限分配
- **动态菜单**：根据用户权限自动过滤可见菜单，支持父子级联结构
- **流程审批**：可视化拖拽建模，审批/抄送/条件分支，版本管理，超时处理
- **入库单管理**：采购入库全流程，材料明细、附件管理、金额自动计算
- **计划任务**：Cron 定时调度，支持 C# 脚本和反射处理器
- **集成平台**：低代码数据同步，HTTP 连接管理，字段映射

## 内置账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 超级管理员 | `admin` | `password` |

内置三种角色：超级管理员（全部权限）、普通用户（基础查看）、只读用户（仅查看）。
